# Assignment 5 — Final Exam Hints

---

## Key Skills to Master

- **Properties and proofs of special matrices**
  - You must understand and be able to prove properties of:
    - Singular matrices  
    - Symmetric matrices  
    - Orthogonal matrices  
  - See **Question 1**.
  - Important facts to remember:
    - A matrix is **singular** if $\det(A) = 0$.
    - A matrix is **orthogonal** if:
$$
A^T A = I.
$$
    - A matrix is **symmetric** if:
$$
A^T = A.
$$

  - You should also be able to **find unknown values** in matrices using these properties.
  - See **Coursework Exam 2 (G1)** and the **practice questions for G1 (CW 2)**.

---

- **Finding the inverse using row reduction**
  - You must be able to compute the inverse of a matrix using row operations.
  - The method involves forming the augmented matrix:
$$
[A \mid I]
$$
  and row reducing to:
$$
[I \mid A^{-1}].
$$
  - See **Question 2**.

- **Theory involving inverses**
  - Be comfortable with theoretical results involving inverses.
  - Understand:
    - When an inverse exists  
    - Relationships between determinants and inverses  
  - See **Question 3 (parts b, c, d, e)**.

---

## Important Questions to Review

- **Question 1** — Proofs and properties of special matrices  
- **Question 2** — Finding inverses using row reduction  
- **Question 3 (b–e)** — Theory involving inverses  

---

## Common Mistakes to Avoid

- Forgetting key definitions (e.g., orthogonal means $A^T A = I$).
- Making algebra errors when solving for unknown entries in matrices.
- Incorrect row operations when finding inverses.
- Not augmenting the identity matrix correctly.
- Assuming a matrix has an inverse without checking conditions.
- Misinterpreting theoretical questions about inverses.
