# Assignment 6 — Final Exam Hints

---

## Key Skills to Master

- **Determinants and notation**
  - You must clearly distinguish between a matrix and its determinant.
  - If 
$$
A = \begin{pmatrix} \cdots \end{pmatrix},
$$
  then its determinant must be written using vertical bars:
$$
\det(A) = |A|.
$$

  - When computing determinants using row reduction:
    - Always use **determinant notation (vertical bars)**.
    - Clearly show each row operation.

- **Row reduction for determinants**
  - You must present your work clearly.
  - Row operations should be written **to the right-hand side** of each step.
  - Be careful:
    - Some row operations change the value of the determinant.
    - You must adjust the determinant accordingly when required.

- **Orthogonal matrices (definition and proofs)**
  - You must be able to clearly define an orthogonal matrix:
$$
A^T A = I.
$$

  - Be able to prove basic properties of orthogonal matrices, such as:
    - Columns (and rows) are orthonormal
    - The inverse satisfies:
$$
A^{-1} = A^T.
$$

  - See **Question 2(a)**.

- **Question 2(a)** — Orthogonal matrices and proofs

## Common Mistakes to Avoid

- Using matrix notation instead of determinant notation when calculating determinants.
- Forgetting how row operations affect the determinant.
- Not showing row reduction steps clearly.
- Writing incomplete or unclear proofs for orthogonal matrices.
- Forgetting that for orthogonal matrices, $A^{-1} = A^T$.
